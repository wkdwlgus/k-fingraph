# Troubleshooting Log

> 마주친 문제와 해결을 시간순 기록.
> 형식: 증상 → 원인 → 해결 → 교훈.

## Template

### YYYY-MM-DD: [한 줄 제목]

**증상**
무엇이 어떻게 잘못 동작했는가. 에러 메시지, 재현 절차.

**원인**
근본 원인. 표면적 원인 말고.

**해결**
무엇을 했는가. 코드 변경, 설정 변경, 우회.

**교훈**
다시 만나지 않으려면 무엇을 기억해야 하는가.
관련 anti-pattern을 `CLAUDE.md`에 추가했는지 체크.
